# Matrix Finance TUI
Zaawansowany, terminalowy menedżer finansów osobistych napisany w Pythonie (Textual).

## Funkcje:
* Hakerski interfejs TUI (Text User Interface)
* Obsługa kont, kategorii, skarbonek i płatności cyklicznych
* Szyfrowanie kopii zapasowych algorytmem AES-256 (Kryptografia)
* Zaawansowane raporty z eksportem do TXT/CSV

## Jak uruchomić:
1. Zainstaluj wymagania: `pip install -r requirements.txt`
2. Uruchom plik główny: `python main.py`
